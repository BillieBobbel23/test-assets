Test-assets

A tiny, self contained package with front-end assets.
Useful for testing and debugging automations scripts and other frontend tools.

Installation:

git clone https://github.com/BillieBobbel23/test-assets

Contents:

  • index.html, a simple landing page
  • style.scss, a sass file with some styles
  • Lato typeface is used in regular and bold variant
  • Some acid cat images, you know... because cats
